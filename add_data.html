Adding data streams to your Data Pool

A stream contains a configured data set that you would like to compare in Data Pool. Streams are needed to create projects. Select a stream on the left, and the filters you'd like to use. Then add it to your stream list.

Once it's added, you can configure it some more on the right. You don't need to completely define everything, it's also possible to configure on the fly in the Data Pool visualisation dashboard.

Don't forget to save once you're done. Then head over to the Projects section.
